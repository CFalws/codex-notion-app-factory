# Momentum Timer Spec

## Overview

Momentum Timer is a local personal-use app that reduces task-start friction by combining a concrete first action with an immediate 3-minute countdown.

## Core User Flow

1. User enters a vague task
2. App generates one concrete first action
3. App immediately starts a 3-minute momentum timer
4. User focuses only on starting, not planning

## Key Features

- single text input
- one-action output
- immediate 3-minute countdown
- recent launch history

## Constraints

- local-only
- minimal UI
- instant interaction

## Success Criteria

- user receives a first action within seconds
- timer starts without extra clicks
- interface feels lighter than a full to-do app

## V1 Scope

- task input
- first-action generator
- countdown display
- local history
