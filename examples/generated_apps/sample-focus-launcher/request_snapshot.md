# Request Snapshot

## Source

Simulated structured intake request using the request contract in `templates/app_request.md`.

## Project Name

Focus Launcher

## Problem

I procrastinate when starting small chores because normal task managers create more overhead than momentum.

## Desired Outcome

A tiny personal tool that gives me one concrete first action instead of a large task list.

## Primary User

Myself

## Preferred Stack

Python backend with a very small browser UI.

## Constraints

- should load instantly
- should not feel like a project manager
- should work well for single-user local usage

## Definition of Done

- one task input
- one concrete first action
- recent history

## Mode

`build`
