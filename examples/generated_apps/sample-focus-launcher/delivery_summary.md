# Delivery Summary

## Request

Build a tiny personal tool that helps me start chores by giving me exactly one concrete first action instead of a big task list.

## Produced Output

- implementation-oriented spec
- implementation plan
- execution checklist
- working local web app

## App Behavior

The app accepts a raw task such as:

- clean the kitchen
- reply to emails
- start studying math

It returns one concrete first action such as:

- Put one dirty dish in the sink and clear a single counter spot.
- Open your inbox and reply to the shortest unanswered email.
- Open your notes and solve just the first problem on the page.

## Why This Matters

This sample shows the intended pipeline shape:

structured request -> Codex planning artifacts -> implemented local app
