# Focus Launcher Spec

## Overview

Focus Launcher is a tiny personal-use app that turns a vague chore into a single frictionless starting action.

## Core User Flow

1. User enters a chore such as “clean the kitchen”
2. App converts it into one small starting action
3. User starts immediately instead of planning

## Key Features

- single-input task capture
- one-action output
- low-friction interface
- recent history

## Constraints

- must be very fast
- must avoid overwhelming UI
- optimized for solo personal use

## Success Criteria

- user can get a first action within seconds
- interface feels lighter than a normal to-do app

## V1 Scope

- one text input
- one suggested first action
- short history list
